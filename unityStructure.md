# File Structure
===
Assets/
: Carpeta de Unity donde se encuentran todos los recuersos a utilizar para el desarrollo del videojuego
Scenes/
: Carpeta donde se encuentran las escenes .unity
Scripts/
: Carpeta donde se encontrar치n todos los scripts del videojuego
Resources/
: Carpeta donde se encuentran recursos como: audios, modelos, prefabs, sprites
Prefabs/
: Carpeta para los objetos prefabricados
Sprites/
: Carpeta para los sprites del videojuego que no seran optimizados

```
Assets
|
|-- Scenes
|    |-- [ScenesFolder]
|           |-- Scene.unity
|
|-- Scripts
|    |-- [sceneFolder]
|    |-- [generalFolder]
|
|-- Resources (minigames)
|    |-- [audioFolder]
|    |      |-- [sceneFolder]
|    |-- [spriteFolder]
|    |      |-- [sceneFolder]
|    |-- [prefabFolder]
|           |-- [sceneFolder]
|
|-- Prefabs
|    |-- [sceneFolder]
|    |-- [generalFolder]
|
|-- Sprites
|    |-- [sceneFolder]
|    |-- [generalFolder]
```
