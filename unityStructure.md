# File Structure

Assets
: Carpeta de Unity donde se encuentran todos los recuersos a utilizar para el desarrollo del videojuego


```
Assets
|
|-- Scenes
|    |-- [ScenesFolder]
|           |-- Scene.unity
|
|-- Scripts
|    |-- [sceneFolder]
|    |-- [generalFolder]
|
|-- Resources (minigames)
|    |-- [audioFolder]
|    |      |-- [sceneFolder]
|    |-- [spriteFolder]
|    |      |-- [sceneFolder]
|    |-- [prefabFolder]
|           |-- [sceneFolder]
|
|-- Prefabs
|    |-- [sceneFolder]
|    |-- [generalFolder]
|
|-- Sprites
|    |-- [sceneFolder]
|    |-- [generalFolder]
```
