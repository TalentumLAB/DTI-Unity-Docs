# File Structure

```
Assets
|
|-- Scenes
|    |-- [ScenesFolder]
|           |-- Scene.unity
|
|-- Scripts
|    |-- [componentFolder]
|    |-- [generalFolder]
|
|-- Resources (minigames)
|    |-- [audioFolder]
|    |      |-- [componentFolder]
|    |-- [spriteFolder]
|    |      |-- [componentFolder]
|    |-- [prefabFolder]
|           |-- [componentFolder]
|
|-- Prefabs
|    |-- [componentFolder]
|    |-- [generalFolder]
|
|-- Sprites
|    |-- [componentFolder]
|    |-- [generalFolder]
```
